Бот телеграмм для нахождения оригинального номера запасной части по вин коду.

пользователь вводит в бот вин код от автомобиля.
бот вставляет этот вин код в строку поиска на сайте autodoc.ru
возвращает пользователю данные по его машине : марку, модель, год выпуска.
бот предлагает пользователю меню для выбора запчасти: 
фильтр масляный
фильтр воздушный
фильтр салона

пользователь выбирает из предложенных вариантов.
бот запрашивает на сайте autodoc.ru каталожный номер этой запчасти и возвращает его в чат пользователю
